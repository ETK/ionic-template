Ionic-template
=====================

Starting projects for Ionic.

## Using this project

#####Install ionic

```bash
$ sudo npm install -g ionic
```

More info on this can be found on the Ionic [Getting Started](http://ionicframework.com/getting-started) page.


## Testing

#####Run in browser

```bash
$ cd projectName
$ sudo ionic serve
```

or 

```bash
$ ionic serve --lab
```

#####Run on device

Run on iOS device 

```bash
$ cd projectName
$ sudo ionic run ios
```

Run on Android device
```bash
$ cd projectName
$ sudo ionic run android
```
